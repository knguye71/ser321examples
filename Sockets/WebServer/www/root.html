You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=x&num2=y -- returns x*y (replace x and y with integers)
  • /github?query=users/example/repos -- returns public repos of user example
  • /append?string1=x&string2=y -- returns appended string of string1 and string 2
  • /contain?string1=abc&string2=a -- returns true or false if string1 contains string2
